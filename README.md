# Tailwind CSS Presentation

This project is a responsive informational website about Tailwind CSS, created for a web development assignment. The website explains what Tailwind CSS is, why developers use it, how it works, installation methods, real-world usage, and future trends.

## Project Overview

The website is built with HTML, CSS, and JavaScript. It uses Tailwind CSS through the browser CDN for utility classes and includes custom project files for page structure, normalization, and mobile navigation behavior.

## Topics Covered

- What Tailwind CSS is
- History of Tailwind CSS
- Why developers use Tailwind CSS
- Advantages and disadvantages
- How to use Tailwind utility classes
- Regular CSS vs Tailwind CSS
- Installation methods: CDN and CLI
- Real-world industry usage
- AI and future frontend development trends
- Official resources for learning more

## Assignment Requirements

This project includes:

- A website created with HTML, CSS, and JavaScript
- Content explaining the technology, what it does, why it is used, where it is used, and how to use it
- Responsive design for desktop and mobile screens
- `img` with `srcset` for responsive image loading
- `picture` with `source` for responsive media markup
- Navigation links for moving through the page sections
- External links to official documentation and related resources

## Files

- `index.html` - Main website content and page structure
- `styles/normalize.css` - Browser style normalization
- `styles/styles.css` - Custom CSS
- `scripts/scripts.js` - Mobile menu interaction
- `images/` - Images, framework logos, and enterprise logos used in the website

## How to View

Open `index.html` in a web browser.

Because Tailwind CSS is loaded through a CDN script, an internet connection is needed for the Tailwind utility classes to load correctly.
